# Calculator

A simple desktop calculator built using **Java** and **JavaFX**.
The application performs basic arithmetic operations through a graphical user interface defined with **FXML**.

The project is organized into separate packages for the application entry point, controller, and calculation logic.

---

## Features

* Addition
* Subtraction
* Multiplication
* Division
* JavaFX graphical interface
* Organized project structure

---

## Project Structure

```text
calculator
│
├── app
│   ├── App.java
│   └── Launcher.java
│
├── controller
│   └── CalculatorController.java
│
├── model
│   └── Calculator.java
│
├── service
│   └── Operations.java
│
└── resources
    └── folders
        └── calculator
            └── view.fxml
```

* **App** – JavaFX application class
* **Launcher** – Entry point used to launch the JavaFX application
* **Controller** – Handles user interactions
* **Model / Service** – Contains the calculation logic
* **FXML** – Defines the graphical interface
